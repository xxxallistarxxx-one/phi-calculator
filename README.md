**Phi-Ratio Geometry Calculator** - Advanced tool for Symmetric Field Physics.

Calculates complete phi-ratio (golden ratio) series and related sacred geometry proportions.

- Phi-ratio series calculation in both directions
- Resonant Frequency calculations
- Sacred Geometry & Platonic Solids tools
- Great Pyramid of Giza proportions

1. Enter any single dimension (diameter, height, gap, radius, etc.)
2. Choose a unit and the number of steps
3. Click Calculate
4. Get full phi-ratio series and related geometric values
